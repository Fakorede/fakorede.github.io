## Overview

My primary research focuses on developing static analysis techniques for Android application security, with a particular emphasis on:

- **Android Auto Compliance**: Detecting violations of safety guidelines in automotive apps
- **Taint Analysis**: Using FlowDroid and SootUp for information flow analysis
- **Callback-Aware Analysis**: Handling Android's event-driven programming model

## Tools & Frameworks

- **SootUp**: Next-generation static analysis framework for Java/Android
- **FlowDroid**: Precise taint analysis for Android applications
- **Soot**: Classic static analysis framework
